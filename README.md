# MC Server Controller Discord Bot

Made with TS, Discord.js

## Commands

Install dependencies

```bash
yarn install
```

Run develop

```bash
yarn start:dev
```

Build for production

```bash
yarn build
```

Refresh slash commands in Server

```bash
yarn refresh:slash
```
